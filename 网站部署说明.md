# 网站部署说明 - 让别人访问你的网站

## 🌐 方法一：GitHub Pages（推荐，免费）

### 步骤 1：启用 GitHub Pages

1. **访问你的仓库**
   - 打开：https://github.com/jojodd77/cosmic-resonance
   - 点击顶部的 **"Settings"**（设置）标签

2. **找到 Pages 选项**
   - 在左侧菜单中找到 **"Pages"** 选项
   - 点击进入

3. **配置 Pages**
   - 在 **"Source"** 下拉菜单中选择 **"main"** 分支
   - 在 **"Folder"** 中选择 **"/ (root)"**
   - 点击 **"Save"** 按钮

4. **等待部署**
   - 等待 1-2 分钟，GitHub 会自动部署你的网站
   - 部署完成后，会显示你的网站地址：
     ```
     https://jojodd77.github.io/cosmic-resonance/
     ```

5. **分享链接**
   - 将这个链接分享给任何人，他们就可以访问你的网站了！

### 特点
- ✅ 完全免费
- ✅ 自动部署（每次推送代码自动更新）
- ✅ 使用 HTTPS（安全）
- ✅ 全球 CDN 加速
- ✅ 无流量限制

### 注意事项
- 仓库必须是 Public（公开）才能免费使用
- 如果是 Private 仓库，需要 GitHub Pro 账户

## 🚀 方法二：Netlify（免费，更灵活）

### 步骤

1. **访问 Netlify**
   - 打开：https://www.netlify.com
   - 使用 GitHub 账号登录

2. **部署网站**
   - 点击 **"Add new site"** → **"Import an existing project"**
   - 选择 **"Deploy with GitHub"**
   - 授权 Netlify 访问你的 GitHub
   - 选择仓库：`jojodd77/cosmic-resonance`
   - 点击 **"Deploy site"**

3. **获得链接**
   - 部署完成后，会得到一个随机域名，例如：
     ```
     https://cosmic-resonance-123abc.netlify.app
     ```

4. **自定义域名（可选）**
   - 在站点设置中可以绑定自己的域名
   - 例如：`cosmic-resonance.com`

### 特点
- ✅ 免费
- ✅ 自动部署
- ✅ 可以自定义域名
- ✅ 支持私有仓库
- ✅ 更快的部署速度

## ⚡ 方法三：Vercel（免费，适合前端项目）

### 步骤

1. **访问 Vercel**
   - 打开：https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 **"Add New..."** → **"Project"**
   - 选择仓库：`jojodd77/cosmic-resonance`
   - 点击 **"Deploy"**

3. **获得链接**
   - 部署完成后，会得到一个域名，例如：
     ```
     https://cosmic-resonance.vercel.app
     ```

### 特点
- ✅ 免费
- ✅ 自动部署
- ✅ 全球 CDN
- ✅ 可以自定义域名

## ⚠️ 常见问题

### Q: 网站显示 404 错误？
A: 
- 检查 GitHub Pages 是否已启用
- 确认分支选择正确（main）
- 等待几分钟后重试（部署需要时间）

### Q: 网站更新后看不到变化？
A:
- 清除浏览器缓存（Ctrl+F5 或 Cmd+Shift+R）
- 等待几分钟（CDN 缓存更新需要时间）

### Q: 可以绑定自己的域名吗？
A:
- 可以！在 GitHub Pages 设置中可以添加自定义域名
- 需要在域名 DNS 中添加 CNAME 记录
