# 本地开发测试指南

## 🎯 开发流程

```
本地修改代码 → 本地预览测试 → 确认无误 → 推送到 GitHub → 自动部署到 Vercel
```

---

## 🖥️ 本地预览方法

### 方法一：使用启动脚本（最简单）

1. **双击运行脚本**
   - 双击 `启动服务器.sh` 文件
   - 或在终端运行：
     ```bash
     cd "/Users/jojodd/Desktop/1.0算命前端"
     ./启动服务器.sh
     ```

2. **访问本地网站**
   - 脚本会自动启动服务器
   - 在浏览器访问：`http://localhost:8000`
   - 或：`http://127.0.0.1:8000`

3. **停止服务器**
   - 在终端按 `Ctrl + C`

---

### 方法二：使用 Python（推荐）

#### macOS/Linux:

```bash
cd "/Users/jojodd/Desktop/1.0算命前端"
python3 -m http.server 8000
```

#### Windows:

```bash
cd "C:\Users\jojodd\Desktop\1.0算命前端"
python -m http.server 8000
```

然后访问：`http://localhost:8000`

---

### 方法三：使用 Node.js

```bash
cd "/Users/jojodd/Desktop/1.0算命前端"

# 方法 1：使用 npx serve（无需安装）
npx serve .

# 方法 2：使用 http-server（需要先安装）
npm install -g http-server
http-server -p 8000
```

---

### 方法四：使用 VS Code Live Server（推荐开发使用）

1. **安装扩展**
   - 在 VS Code 中搜索并安装 "Live Server" 扩展

2. **启动**
   - 右键点击 `index.html`
   - 选择 "Open with Live Server"
   - 自动在浏览器打开，通常是：`http://localhost:5500`

3. **优势**
   - ✅ 自动刷新（修改代码后自动更新）
   - ✅ 热重载
   - ✅ 非常方便开发

## 🔄 开发工作流

### 日常开发流程：

1. **启动本地服务器**
   ```bash
   cd "/Users/jojodd/Desktop/1.0算命前端"
   python3 -m http.server 8000
   ```

2. **在浏览器打开**
   - 访问：`http://localhost:8000`
   - 保持浏览器打开

3. **修改代码**
   - 使用编辑器修改 HTML、CSS、JavaScript 文件
   - 保存文件

4. **刷新浏览器查看效果**
   - 按 `F5` 或 `Cmd+R` 刷新
   - 或使用 Live Server（自动刷新）

5. **测试功能**
   - 测试所有功能是否正常
   - 在不同浏览器中测试
   - 检查控制台是否有错误

6. **确认无误后部署**
   ```bash
   git add .
   git commit -m "描述你的修改"
   git push origin main
   ```
   - Vercel 会自动部署更新

## 🛠️ 开发工具推荐

### 浏览器开发者工具

打开方式：
- **Chrome/Edge**: `F12` 或 `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows)
- **Firefox**: `F12` 或 `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows)
- **Safari**: 需要先启用开发者菜单

常用功能：
- **Console**: 查看 JavaScript 错误和日志
- **Elements**: 检查 HTML 和 CSS
- **Network**: 查看资源加载情况
- **Application**: 查看存储和缓存

## 📝 本地 vs 线上

### 本地预览（开发环境）

- **地址**: `http://localhost:8000`
- **用途**: 开发、测试、调试
- **特点**: 
  - ✅ 修改立即生效
  - ✅ 不需要网络（除了 CDN 资源）
  - ✅ 可以快速迭代
  - ❌ 只有你能访问

### 线上部署（生产环境）

- **地址**: `https://cosmic-resonance.vercel.app`（或你的 Vercel 域名）
- **用途**: 展示给其他人
- **特点**:
  - ✅ 任何人都可以访问
  - ✅ 全球 CDN 加速
  - ✅ 自动 HTTPS
  - ❌ 需要推送代码才能更新

## 🎨 开发技巧

### 1. 使用浏览器自动刷新

**Live Server**（VS Code 扩展）：
- 修改代码后自动刷新浏览器
- 无需手动刷新

**BrowserSync**（命令行工具）：
```bash
npm install -g browser-sync
cd "/Users/jojodd/Desktop/1.0算命前端"
browser-sync start --server --files "**/*"
```

### 2. 多设备测试

如果电脑和手机在同一网络：

1. **找到电脑的 IP 地址**
   ```bash
   # macOS/Linux
   ifconfig | grep "inet "
   
   # Windows
   ipconfig
   ```

2. **在手机上访问**
   ```
   http://你的IP地址:8000
   ```
   例如：`http://192.168.1.100:8000`

### 3. 清除缓存

开发时如果看不到更新：
- **强制刷新**: `Cmd+Shift+R` (Mac) / `Ctrl+F5` (Windows)
- **清除缓存**: 浏览器设置 → 清除浏览数据
- **无痕模式**: 使用无痕窗口测试

## 🔍 调试技巧

### 检查控制台错误

1. 打开开发者工具（F12）
2. 查看 Console 标签页
3. 红色错误信息表示有问题
4. 根据错误信息修复代码

### 检查网络请求

1. 打开开发者工具
2. 查看 Network 标签页
3. 刷新页面
4. 查看哪些资源加载失败（红色标记）

### 检查元素样式

1. 打开开发者工具
2. 使用元素选择器（左上角图标）
3. 点击页面元素
4. 在右侧查看和修改 CSS
